Description

Allows users to track their reading. Three shelves exisit: To Read, Reading, and Read. Books are searched for using a form that performs requests to the Google Book API. Book views and searches all have data retreived from Google's services. Bookmarks are how users save books. A bookmark stores information to quickly retreive books from Google's Book API for future requests.

Functionality

Basic functionality will include user authentication using devise. Book search using Google's Book API. Book management, including creation, deletion, and editing of bookmarks. Shelf views to display user's bookmarks.
